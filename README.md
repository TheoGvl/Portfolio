# Personal Portfolio - Theofilos Geivelis

A modern, responsive personal portfolio website built to showcase my academic journey, technical skills, and professional certifications. 

## Features
* **Profile Picture:** A circular profile picture integrated with the dark theme's accents.
* **Dark Theme Design:** A sleek interface utilizing a dark background with vibrant purple and pink accents.
* **Interactive UI:** Clickable accordion menus to cleanly organize and display certificates by issuer IBM Cognitive Class, Anthropic-Claude, and SUCReD.
* **Responsive Layout:** Fully styled with custom CSS for readability and a modern aesthetic.

## Technologies Used
* **HTML5:** Semantic structure and content organization.
* **CSS3:** Custom styling, CSS Grid, and interactive hover effects.
* **Vanilla JavaScript:** Logic for toggling the certificate dropdown panels.

## Project Structure
* `index.html`: The main HTML document containing the portfolio structure and content.
* `style.css`: The stylesheet defining the dark theme, colors, layout, and profile picture styling.
* `image_1.webp`: The personal profile picture used in the header.
