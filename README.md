# Personal Portfolio - Theofilos Geivelis

A modern, responsive personal portfolio website built to showcase my academic journey, technical skills, and professional certifications.

## Features

*   **Profile Picture:** A circular profile picture integrated with the dark theme's soft purple accents.
*   **Dark Theme Design:** A sleek interface utilizing a dark background with vibrant soft purple and neon pink accents.
*   **Custom Scrollbar:** A slim, themed scrollbar with a soft purple handle for a cohesive browsing experience.
*   **Visual Skill Badges:** A modern grid of glowing, interactive badges showcasing self-taught technical skills.
*   **Social Connectivity:** Integrated footer with professional icons linking to GitHub, LinkedIn, and Email.
*   **Interactive UI:** Clickable accordion menus to cleanly organize and display certificates by issuer (IBM Cognitive Class, Anthropic-Claude, and SUCReD).
*   **Scroll Animations:** Sections that smoothly fade and slide into view as the user scrolls.
*   **Responsive Layout:** Fully styled with custom CSS for readability and a modern aesthetic.

## Technologies Used

*   **HTML5:** Semantic structure and content organization.
*   **CSS3:** Custom styling, CSS Grid, custom scrollbar properties, and interactive neon hover effects.
*   **Vanilla JavaScript:** Logic for toggling the certificate dropdown panels and Intersection Observer API for scroll animations.
*   **FontAwesome:** Professional iconography for social links and skill badges.

## Project Structure

*   **index.html:** The main HTML document containing the portfolio structure, content, and animation logic.
*   **style.css:** The stylesheet defining the dark theme, color variables, custom scrollbar, and badge styling.
*   **image_1.webp:** The personal profile picture used in the header.
